# LinkMux
A url shortener project.

# 需求
- PHP >= 7.1.3
- PHP Extension
    - OpenSSL PHP Extension
    - PDO PHP Extension
    - Mbstring PHP Extension
    - Tokenizer PHP Extension
    - XML PHP Extension
    - Ctype PHP Extension
    - JSON PHP Extension
    - BCMath PHP Extension
- [Composer](https://getcomposer.org/)
- node >= v9.9.0
